
The goal of this project is to perform sentiment analysis on tweets. 

A. Steps 1 and 2 comprise of how to search for tweets and how to compute the term frequency of the tweets.

B. Steps 3,4 and 5 comprise of how to find the sentiment scores of (tweets, tweets based on username, and tweets based on location/origin of the tweet) respectively.

C. Please find 'Instructions.txt' to run the different functionalities of the project.

D. Please find the 'Report.txt' which will include all the results that you will obtain. It will be helpful to cross verify the output.(NOTE: you output will not be similar)
